# 一个Vue3项目便捷开发模板

使用该模板搭建项目，因为配置好了很多常用的功能，所以可以快速开发页面，而无需关注基础工具的搭建，从而节省了开发时间。

## 本模板配置了以下功能

1. vite3
2. vue3
3. vue-router4
4. pinia
5. axios
6. sass
7. mitt
8. typescript
9. eslint
10. prettier
11. svg-icon

## 使用方法
```shell
npx degit plas007/LJVue3Template#master 项目名称
cd 项目名称
npm install
npm run serve
```
